WHAT IT FIXES:
──────────────────────────────────────────
✓ com.brother-paper-size-error

KEY DISCOVERY:
──────────────────────────────────────────
The fix is  media=DC16  — Brother's internal PPD code for
DK-1241 102mm x 152mm (4x6") labels. No other media code
works. This is completely undocumented by Brother.
