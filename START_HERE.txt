STEP 1 — RUN THIS FIRST (one time only):
──────────────────────────────────────────
Right-click  INSTALL.command  → Open → click Open

Clears macOS Gatekeeper warnings from all files and
installs Python dependencies. Only needed once.

STEP 2 — UPDATE YOUR DRIVER (recommended):
──────────────────────────────────────────
Double-click  UPDATE_DRIVER.command

Downloads the latest Brother driver (v1.11.0d, July 2025)
directly from Brother's servers. Fixes signal 11 crashes
on Apple Silicon caused by the old 2018 Intel-only driver.
Skip this if your driver is already from 2021 or newer.

STEP 3 — FIX YOUR PRINTER:
──────────────────────────────────────────
Double-click  RUN_FIX.command
Enter your Mac password when asked.
Watch the printer — it prints a test label when done.

ALL THE FILES:
──────────────────────────────────────────
⭐ INSTALL.command          ← RUN FIRST (clears security)
⭐ UPDATE_DRIVER.command    ← Update to 2025 ARM64 driver
⭐ RUN_FIX.command          ← Full fix + test print
   TEST_PRINT.command       ← Send a test label
   DIAG_ONLY.command        ← Diagnose without changes
   SHIPPING_LABEL.command   ← Print a shipping label
   CLEAN_PTOUCH.command     ← Fix P-touch Editor duplicates
   WATCH_FOLDER.command     ← Auto-print PDFs from a folder

WHAT IT FIXES:
──────────────────────────────────────────
✓ Old 2018 Intel-only driver on Apple Silicon (M1-M4)
✓ rastertobrotherQL1100 crashed on signal 11
✓ com.brother-paper-size-error loop
✓ Stuck print jobs / "Filter failed"
✓ Duplicate entries in P-touch Editor

KEY DISCOVERY:
──────────────────────────────────────────
The fix is  media=DC16  — Brother's internal PPD code for
DK-1241 102mm x 152mm (4x6") labels. No other media code
works. This is completely undocumented by Brother.

COMPATIBLE LABELS:
──────────────────────────────────────────
✓ COLORWING DK-1241 compatible rolls
✓ Genuine Brother DK-1241 rolls
✓ Any 102mm x 152mm 4x6" compatible roll

Tested: Mac Studio M4 · macOS 26 · Sequoia
